# Systematic Review Package (A1)

This folder provides reproducible A1 artifacts aligned with `作业要求.md`:
- protocol
- search strategy
- inclusion/exclusion rules
- PRISMA flow log
- structured extraction table
- quality assessment table

Scope:
- Topic: multimodal ICU prediction with time-series + clinical notes
- Core targets: datasets, tasks, alignment units/windows, model families, evaluation metrics

Status:
- Current starter-reference set (n=5) has completed extraction and quality scoring.
- Included studies are explicitly tagged for comparability in `study_extraction.csv`.
- Non-comparable but method-relevant studies are kept with `INCLUDE_LIMITED_COMPARABILITY` for transparency.

Notes:
- This package is version-controlled and can be extended with additional screened studies.
- Tables are plain-text CSV for auditability and reproducibility.
