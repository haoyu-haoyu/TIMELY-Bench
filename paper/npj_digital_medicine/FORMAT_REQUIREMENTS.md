# npj Digital Medicine Format Notes

This submission bundle targets `npj Digital Medicine` Article format.

Primary sources checked on 2026-04-19:

- `npj Digital Medicine` content types:
  - https://www.nature.com/npjdigitalmed/content-types
- `npj Digital Medicine` for authors:
  - https://www.nature.com/npjdigitalmed/for-authors-and-referees
- `npj Digital Medicine` submission page:
  - https://www.nature.com/npjdigitalmed/for-authors-and-referees/submisions
- Springer Nature support note on LaTeX template:
  - https://support.springernature.com/en/support/solutions/articles/6000127538-submit-a-latex-manuscript-to-a-springer-journal-using-overleaf

## Hard constraints applied in this bundle

- Article title:
  - target `<= 15` words
  - avoid punctuation, idioms, and puns
- Abstract:
  - no subheadings
  - target `<= 150` words
- Introduction:
  - no subheadings
- Results:
  - subheadings allowed
- Discussion:
  - no subheadings
  - no standalone `Limitations` section
  - no standalone `Conclusions` section
- Methods:
  - subheadings allowed
- Figures:
  - each figure kept as a separate asset file
  - multi-panel figures should remain a single combined image file
- Initial submission:
  - Nature states strict formatting is not required at first submission
  - this bundle still follows the acceptance-stage structure so later conversion cost is lower

## Inference used in template choice

The `npj Digital Medicine` pages do not explicitly state a reference-style file.
For the LaTeX bundle, `sn-jnl` with the `sn-nature` option is used as a
pragmatic Nature Portfolio style inference based on the official Springer
Nature LaTeX template distributed via Overleaf.

## Local implementation notes

- `figure1_final.png` is the dedicated Figure 1 asset path used by the TeX
  source.
- If the user-supplied final Figure 1 raster is later saved to this exact path,
  no TeX changes are required.
- Main tables are generated from frozen CSV outputs under:
  - `results/cres_v3/paper_tables/`
- Table fragments are generated by:
  - `code/v3/build_npj_submission_tables.py`

## Remaining manuscript placeholders

- author list
- affiliations
- corresponding author email
- acknowledgements and funding text
- data availability statement
- code availability statement
- competing interests statement
- reference list and in-text citations
